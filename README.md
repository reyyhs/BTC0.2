# BTC0.2
Installation
1. Install Python new version
2. Install GIT :
```bash
apt Install git
```
3. Download script manual or use git :

```bash
git clone https://www.github.com/reyyhs/BTC0.2
```

4. Make sure you already in bot folder :

```bash
CD BTC0.2
```

5. Install Module :

```bash
pip install bitcoinlib requests tqdm
```

6. Run Command :

```bash    
python run.py
```
